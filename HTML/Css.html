CSS Basics and Stylesheet Types

CSS

CSS, which stands for Cascading Style Sheets, is used to style and design web pages. It helps control how elements on a page look, including colors, fonts, spacing, layout, and more. With CSS, you can make your website visually appealing and easy to use. It works alongside HTML, which provides the structure, while CSS focuses on the presentation. Using CSS, you can create responsive designs that work well on different screen sizes and devices.

ðŸŽ¯ Basic CSS Properties

  • color: Sets the text color.
  • font-size: Sets the size of the font.
  • font-family: Sets the font style (e.g., Arial, Times New Roman).
  • background-color: Sets the background color of an element.
  • margin: Controls the space outside the element.
  • padding: Controls the space inside the element.
  • border: Adds a border around the element.
  • width: Sets the width of the element.
  • height: Sets the height of the element.
  • text-align: Aligns the text (left, center, right).
  • display: Sets how elements are displayed (block, inline, flex, etc.).
  • position: Sets positioning (static, relative, absolute, fixed).
  • overflow: Controls overflow behavior (hidden, scroll, auto).
  • z-index: Controls stack order of elements.

This paragraph uses internal CSS. Styles are defined in the <style> tag inside the HTML document's head.

3. External CSS

This paragraph uses external CSS. Styles are defined in a separate file (styles.css) and linked with <link>.
